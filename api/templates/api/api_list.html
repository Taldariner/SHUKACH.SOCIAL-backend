API Reference

SHUKACH.SOCIALS utilizes REST principles to implements it's API. Thus, this API is accessible through regular web-requests and URLs similar to regular site pages. This page contains description of API endpoints, provided by SHUKASH.SOCIAL.

Rate Limiting

The speed limits for API endpoints are similar to those for website pages and amount to 20 requests per 10 seconds from one address.
